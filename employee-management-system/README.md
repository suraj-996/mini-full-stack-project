### A mini project where you can perform all the crud operation.

## Features
* See all the employee details
* Add the employee details
* Update the employee details
* Delete the employee details

## Tech Stack

### Frontend

* React JS
* HTML
* CSS

### Backend

* Java
* Spring Framework
* Spring Boot
* Spring Data JPA
* MySQL
